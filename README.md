   - 글 Create, Read

   - 좋아요
   - 댓글 Create
- 회원가입
   - 닉네임
   - 아이디
   - 비밀번호

   - 생년월일
   - 성별

   - 비밀번호

- 마이페이지
   - 글 목록 Read(내글만)
   - 프로필사진
   - 상태메세지
- 채팅

   - 채팅
